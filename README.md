# Lavandaria-gestao-documentos

Este projeto foi criado para organizar a gestão de uma lavandaria, incluindo planeamento, contactos, atas, relatórios e requisitos. O objetivo é melhorar a organização diária e facilitar o trabalho da equipa.

## Estrutura de Pastas

- atas
- anexos
- comunicacao
- planeamento
- relatorios

## Como Contribuir

1. Manter os ficheiros organizados.
2. Escrever informação clara e correta.
3. Atualizar documentos sempre que necessário.
